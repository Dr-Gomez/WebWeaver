# WebWeaver

WebWeaver is a robust and versatile game engine designed specifically for developing 2D platform-fighting games that run in web browsers.

## Features

- **Cross-Platform Compatibility**: Build games that function flawlessly on any device equipped with a web browser.
- **Custom Build System**: Employ a lightweight build system crafted specifically for WebWeaver modifications.
- **Single File Output**: Utilize a built-in JavaScript bundler (serve.py) to generate a single HTML file as the final output.
- **Modular Architecture**: Easily extend and customize the engine to meet your goals using mods.
- **Intuitive API**: Benefit from a clear and straightforward API for fast development.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- The [Python interpreter](https://www.python.org/downloads/) (usually pre-installed with most Linux distributions)
- The [Git Version Control system](https://git-scm.com/downloads) (for cloning the repository)


### Setting Up Your Project

1. Create a new directory for your game project:
    ```bash
    mkdir my-game
    cd my-game
    ```
2. Clone the WebWeaver:
    ```bash
    git clone https://github.com/Dr-Gomez/WebWeaver.git
    ```
3. Serve your Web Weaver:
    ```bash
    python serve.py
    ```
